安装

  1. 越狱你的Kindle：中文教程 & English
  2. 安装KUAL启动器
  3. 下载最新版本，解压后将文件夹改名为kykky并复制到Kindle中的/mnt/us/extensions/目录下
  4. 点击“开启阅读统计”并重启Kindle
  5. 点击“查看阅读统计”即可

说明

  1. 本插件在Kindle Paperwhite3，Firmware v5.8.7测试通过，如果遇到问题请 提交issue
  2. 源码托管在 github ,欢迎讨论
  3. 本插件以totoptlp 的作品 让Kindle实现阅读时间统计 为基础并经过授权，再次感谢
